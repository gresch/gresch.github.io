1 minute read

Introduction yEd is a modeling tool to model different types of graphs, including BPMN diagrams. Contrary to database-oriented modeling tools, yEd is a file-oriented tool. It saves the diagram in an XML file (GraphML). In case one models diagrams with more than 100 elements, consistency checks and spell checks can become cumbersome and time consuming. I prepared an R script that tries to reduce the effort by fetching the names and types of the modeled BPMN elements and make them available as a spreadsheet file.
